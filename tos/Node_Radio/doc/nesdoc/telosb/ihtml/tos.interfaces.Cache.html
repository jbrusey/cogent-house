Interface: tos.interfaces.Cache

interface Cache<typedef t>

A data item cache. The cache does not own the items it caches: there is no allocation/deallocation policy, or notification of eviction. Correspondingly, using references (pointers) as data items can be difficult.

Author:
Rodrigo Fonseca
Philip Levis

Commands
command void flush() Flush the cache of all entries.
command void insert(t item) Inserts an item in the cache, evicting if necessary.
command bool lookup(t item) Return whether the data item is in the cache.
