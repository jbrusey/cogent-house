Interface: tos.interfaces.SendNotifier

interface SendNotifier

The radio stack notifies the entire system that it is about to send a packet. Other modules in the system can call back within this event to modify the packet's metadata. For example, we can setup a module in the system to make every CTP data packet send with low power listening enabled: event void SendNotifier.aboutToSend[AM_COLLECTION_DATA](uint8_t dest, message_t *msg) { call LowPowerListening.setRxSleepInterval(msg, 512); }
