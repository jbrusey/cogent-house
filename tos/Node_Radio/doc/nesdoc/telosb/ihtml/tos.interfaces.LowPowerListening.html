setLocalWakeupInterval

command void setLocalWakeupInterval(uint16_t intervalMs)

Set this this node's radio wakeup interval, in milliseconds. After each interval, the node will wakeup and check for radio activity. Note: The wakeup interval can be set to 0 to indicate that the radio should stay on all the time but in order to get a startDone this should only be done when the duty-cycling is off (after a stopDone).

Parameters:
intervalMs - the length of this node's Rx check interval, in [ms]
