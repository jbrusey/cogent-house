Interface: tos.chips.sht11.SensirionSht11

interface SensirionSht11

SensirionSht11 is the rich interface to the Sensirion SHT11 temperature/humidity sensor.

Author:
Gilman Tolle <gtolle@archrock.com>
Version:
$Revision: 1.4 $ $Date: 2006-12-12 18:23:12 $

Commands
command error_t measureHumidity() Starts a humidity measurement.
command error_t measureTemperature() Starts a temperature measurement.
command error_t readStatusReg() Reads the current contents of the SHT11 status and control register.
command error_t reset() Resets the sensor.
command error_t writeStatusReg(uint8_t val) Writes a new value to the SHT11 status and control register.

Events
event void measureHumidityDone(error_t result, uint16_t val) Presents the result of a humidity measurement.
event void measureTemperatureDone(error_t result, uint16_t val) Presents the result of a temperature measurement.
event void readStatusRegDone(error_t result, uint8_t val) Presents the value of the status register.
event void resetDone(error_t result) Signals that the sensor has been reset.
event void writeStatusRegDone(error_t result) Signals the completion of the status register write.
