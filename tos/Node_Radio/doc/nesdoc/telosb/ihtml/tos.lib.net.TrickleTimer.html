Interface: tos.lib.net.TrickleTimer

interface TrickleTimer

Commands
command void incrementCounter() Increment the counter C.
command void reset() Reset the timer period to L.
command error_t start() Start the trickle timer.
command void stop() Stop the trickle timer.

Events
event void fired() The trickle timer has fired.

Commands - Details

incrementCounter

command void incrementCounter()

Increment the counter C. When an interval ends, C is set to 0.

reset

command void reset()

Reset the timer period to L. If called while the timer is running, then a new interval (of length L) begins immediately.

start

command error_t start()

Start the trickle timer. At boot, the timer period is its maximum value (H). If a protocol requires starting at the minimum value (e.g., fast start), then it should call reset before start.

Returns:
error_t SUCCESS if the timer was started, EBUSY if it is already running, and FAIL otherwise.

stop

command void stop()

Stop the trickle timer. This call sets the timer period to H and C to 0.

Events - Details

fired

event void fired()

The trickle timer has fired. Signaled if C > K.
