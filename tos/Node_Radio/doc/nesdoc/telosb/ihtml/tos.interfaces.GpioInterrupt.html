Interface: tos.interfaces.GpioInterrupt

interface GpioInterrupt

Author:
Jonathan Hui
Joe Polastre Revision: $Revision: 1.5 $ Provides a microcontroller-independent presentation of interrupts

Commands
command error_t disable() Diables an edge interrupt or capture interrupt
command error_t enableFallingEdge()
command error_t enableRisingEdge() Enable an edge based interrupt.

Events
event void fired() Fired when an edge interrupt occurs.

Commands - Details

disable

command error_t disable()

Diables an edge interrupt or capture interrupt

Returns:
SUCCESS if the interrupt has been disabled

enableFallingEdge

command error_t enableFallingEdge()

enableRisingEdge

command error_t enableRisingEdge()

Enable an edge based interrupt. Calls to these functions are not cumulative: only the transition type of the last called function will be monitored for.

Returns:
SUCCESS if the interrupt has been enabled

Events - Details

fired

event void fired()

Fired when an edge interrupt occurs. NOTE: Interrupts keep running until "disable()" is called
