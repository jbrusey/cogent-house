Commands - Details

crc16

command uint16_t crc16(void *buf, uint8_t len)

Compute the CRC-16 value of a byte array.

Parameters:
'void* COUNT(len) buf' A pointer to the buffer over which to compute CRC.
len - The length of the buffer over which to compute CRC.
Returns:
The CRC-16 value.

seededCrc16

command uint16_t seededCrc16(uint16_t startCrc, void *buf, uint8_t len)

Compute a generic CRC-16 using a given seed. Used to compute CRC's of discontinuous data.

Parameters:
startCrc - An initial CRC value to begin with
'void* COUNT(len) buf' A pointer to a buffer of data
len - The length of the buffer
Returns:
The CRC-16 value.
