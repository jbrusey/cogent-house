Interface: tos.interfaces.Random

interface Random

Interface for generating 16-bit and 32-bit random numbers.

Author:
Barbara Hohlt
Date:
March 1 2005

Commands
command uint16_t rand16() Produces a 32-bit pseudorandom number.
command uint32_t rand32() Produces a 32-bit pseudorandom number.

Commands - Details

rand16

command uint16_t rand16()

Produces a 32-bit pseudorandom number.

Returns:
Returns low 16 bits of the pseudorandom number.

rand32

command uint32_t rand32()

Produces a 32-bit pseudorandom number.

Returns:
Returns the 32-bit pseudorandom number.
