generic module AccessibleBitVectorC(uint16_t max_bits)

Generic bit vector implementation. Note that if you use this bit vector from interrupt code, you must use appropriate atomic statements to ensure atomicity. Updated 24/05/2011 to allow direct access to the bit vector

Parameters:
max_bits - Bit vector length.
