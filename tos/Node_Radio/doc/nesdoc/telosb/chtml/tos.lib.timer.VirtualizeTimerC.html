Component: tos.lib.timer.VirtualizeTimerC

generic module VirtualizeTimerC(typedef precision_tag, int max_timers)

VirtualizeTimerC uses a single Timer to create up to 255 virtual timers.

See TEP102 for more details.

Parameters:
precision_tag - A type indicating the precision of the Timer being virtualized.
max_timers - Number of virtual timers to create.
