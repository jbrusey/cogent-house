Component: tos.lib.timer.CounterToLocalTimeC

generic module CounterToLocalTimeC(typedef precision_tag)

CounterToLocalTimeC converts a 32-bit LocalTime to a Counter.

See TEP102 for more details.

Parameters:
precision_tag - A type indicating the precision of the LocalTime and Counter being converted.
