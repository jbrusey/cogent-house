Component: tos.chips.cc2420.CC2420ActiveMessageP

module CC2420ActiveMessageP

Active message implementation on top of the CC2420 radio. This implementation uses the 16-bit addressing mode of 802.15.4: the only additional byte it adds is the AM id byte, as the first byte of the data payload.
