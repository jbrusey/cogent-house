Component: tos.system.BitVectorC

generic module BitVectorC(uint16_t max_bits)

Generic bit vector implementation. Note that if you use this bit vector from interrupt code, you must use appropriate atomic statements to ensure atomicity.

Parameters:
max_bits - Bit vector length.
