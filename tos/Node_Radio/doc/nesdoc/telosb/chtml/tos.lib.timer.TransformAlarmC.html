Component: tos.lib.timer.TransformAlarmC

generic module TransformAlarmC(typedef to_precision_tag, typedef to_size_type, typedef from_precision_tag, typedef from_size_type, uint8_t bit_shift_right)

TransformAlarmC decreases precision and/or widens an Alarm. An already widened Counter component is used to help.

See TEP102 for more details.

Parameters:
to_precision_tag - A type indicating the precision of the transformed Alarm.
to_size_type - The type for the width of the transformed Alarm.
from_precision_tag - A type indicating the precision of the original Alarm.
from_size_type - The type for the width of the original Alarm.
bit_shift_right - Original time units will be 2 to the power bit_shift_right larger than transformed time units.
