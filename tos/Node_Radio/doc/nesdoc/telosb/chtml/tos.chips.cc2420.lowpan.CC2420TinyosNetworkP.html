Component: tos.chips.cc2420.lowpan.CC2420TinyosNetworkP

module CC2420TinyosNetworkP

Fills in the network ID byte for outgoing packets for compatibility with other 6LowPAN networks. Filters incoming packets that are not TinyOS network compatible. Provides the 6LowpanSnoop interface to sniff for packets that were not originated from TinyOS.
