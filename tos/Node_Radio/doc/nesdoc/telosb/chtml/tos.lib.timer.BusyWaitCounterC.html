Component: tos.lib.timer.BusyWaitCounterC

generic module BusyWaitCounterC(typedef precision_tag, typedef size_type)

BusyWaitCounterC uses a Counter to implement the BusyWait interface (block until a specified amount of time elapses). See TEP102 for more details.

See TEP102 for more details.

Parameters:
precision_tag - A type indicating the precision of the BusyWait interface.
size_type - An integer type representing time values for the BusyWait interface.
